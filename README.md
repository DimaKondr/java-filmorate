# java-filmorate
## Template repository for Filmorate project. ##

__*ER-diagramm of project:*__

![PNG-file with ER-diagramm of Filmorate project.](/ERD-filmorate.png)

__Фильмы имеют:__
  * id;
  * название;
  * описание;
  * дату выхода;
  * длительность;
  * жанр;
  * возрастной рейтинг;
  * лайки пользователей.

__Пользователи имеют:__
  * id;
  * e-mail;
  * логин;
  * имя;
  * дату рождения;
  * список друзей.
